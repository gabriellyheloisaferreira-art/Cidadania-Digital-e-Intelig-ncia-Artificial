# Portal EcoDigital: Cidadania Digital e Inteligência Artificial

## 🎯 Objetivo do Projeto
Este site foi desenvolvido como um portal comunitário interativo de conscientização social focado em instruir os cidadãos a identificarem e combaterem mídias manipuladas por inteligência artificial (Deepfakes) e mitigar os efeitos da desinformação automatizada.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica de elementos.
- **CSS3**: Layout moderno baseado em Flexbox, Media Queries para responsividade móvel e variáveis nativas.
- **JavaScript (ES6)**: Manipulação dinâmica do DOM para acessibilidade e validação lógica.

## 🤖 Prompts e Comandos de IA Utilizados
- *"Gere uma estrutura semântica HTML5 para um portal de notícias moderno contendo lead, manchete, barra de pesquisa e um elemento de quiz contra desinformação."*
- *"Crie regras CSS usando variáveis de cores CSS nativas que permitam alternar de forma limpa entre modo claro e escuro usando seletores específicos."*
